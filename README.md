# AI-Powered Code Review Assistant

## What it does
Analyzes code files and provides intelligent feedback on:
- Code quality
- Best practices
- Potential bugs
- Improvements

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- AI: OpenAI API (or Gemini API - free)

## Features (Planned)
- Upload code files for review
- Real-time AI analysis
- Detailed feedback with suggestions
- Support for multiple programming languages
